Using the Jira webhook integration

This service also accepts Jira webhook events. Configure a webhook in your Jira project settings pointing to:

POST {{ request.host_url }}webhook/jira

Supported events: issue_created, issue_updated. Set your WEBHOOK_SECRET env var to match the secret configured in Jira for signature verification. Generated AC is logged server-side — extend app.py to post it back as a comment or to a Slack channel.
